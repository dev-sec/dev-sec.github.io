Ansible joins Hardening Framework

Thursday, Apr 30, 2015

Ansible joins Hardening Framework

Christoph Hartmann

The Hardening Framework provides best-practice security for DevOps by implementing server hardening with DevOps tools. We are happy to announce that with help of Sebastian Gumprich we were able to implement our first Ansible role: ansible-ssh-hardening. Over the last weeks, we worked hard to release version 1.0:

  • Implement ssh hardening to meet our tests
  • Setup test infrastructure with kitchen-ansible
  • Implement travis tests#7
  • Add handlers to restart sshd only when necessary #6
  • Add support for Oracle Linux #2

The module is available via Ansible Galaxy, now.

As a next step, we plan to add support more modules. ansible-os-hardening is already in the works. Stay tuned and follow us on Twitter.
